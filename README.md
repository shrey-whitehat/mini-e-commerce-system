🛒 Mini E-commerce System
📌 Project Description

This is a mini e-commerce web application developed as a college project.
It allows users to browse products, add items to cart, and simulate an online shopping experience.

🚀 Features
- 🛍️ View products
- ➕ Add to cart
- 🧾 Checkout system (basic)
- 👤 User-friendly interface
- 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js 
- Database: MongoDB 

📂 Project Structure
- /frontend → UI files
- /backend → server code
- /database → database files 

🎯 Purpose
This project was created for learning and academic purposes to understand how e-commerce systems work.
